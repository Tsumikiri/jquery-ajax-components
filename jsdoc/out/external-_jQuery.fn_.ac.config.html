"jQuery.fn".ac. config

These configuration options can be used to change how the plugin behaves.

Properties:

Examples

In this code snippet, we set the value of a config property.

$.fn.ac.config.alertAllowEmpty = false;

Adding a new response processor. This processor will take an array of selectors and run jQuery's .slideUp() on each one.

$.fn.ac.config.responseHandlers.slideUp = function(index, selector) {
    $(selector).slideUp();
};
//Example of expected server response: { "slideUp": [ "#div-to-slide-up", "#other-div-to-slide-up" ] }
