(static) $.fn.ac(eventName, filterSelector, ajaxOptions, alertOptions) → {Array}

The plugin function

This is the main function for this plugin. It binds an event to a page element.
Parameters:
Name Type Description
eventName string The name of an event to use for the binding, such as 'click' or 'submit'
filterSelector string A filter selector for delegated binding
ajaxOptions Object The options to use for ajax requests made by this component
alertOptions Object The options to use for alerts created by this component

Initializing an ajax link
