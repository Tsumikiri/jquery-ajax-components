jquery.ajax-components.js

A jQuery plugin that provides a shorthand interface for binding ajax functionality to page elements. Includes support for automatically inserting pre-generated HTML content into the DOM based on server response directives. The primary function for this plugin is jQuery#ac. Check out the Getting Started Guide.
